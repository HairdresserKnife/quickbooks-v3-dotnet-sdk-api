## 📌 Overview

QuickBooks V3 .NET SDK is a .NET software development kit for working with QuickBooks REST API v3 services.

The project is intended for developers building .NET applications that need to interact with QuickBooks services through the v3 REST API. It provides a repository focused on the SDK layer for QuickBooks API integration.

## 🧩 Purpose

This repository is centered on enabling .NET-based integration with QuickBooks REST API v3 services. It is relevant for software projects that need to connect business, accounting, finance, or workflow applications with QuickBooks data and service endpoints.
